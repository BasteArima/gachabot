## 📌 О проекте

**GachaBot** — это интерактивный развлекательный бот с механикой гачи (коллекционирования карточек), системой дуэлей и коллекций. Бот работает одновременно в **Telegram** и **Discord**, используя общую базу данных, что позволяет игрокам бесшовно объединять свои аккаунты (`/link`) и сохранять прогресс на обеих платформах.

Проект создавался с упором на надежность, масштабируемость и чистоту кода.

## ✨ Ключевые особенности (Key Features)

* **Кроссплатформенность:** Единая бизнес-логика обслуживает два независимых Delivery-слоя (Telegram и Discord).
* **Clean Architecture:** Строгое разделение на слои `Delivery` (обработка команд), `Service` (бизнес-логика) и `Repository` (работа с БД).
* **Высокая производительность:** Использование **Redis** для хранения временных стейтов (кулдауны, сессии дуэлей, квизы, токены привязки аккаунтов), что снижает нагрузку на основную БД.
* **Глубокие игровые механики:**
  * Алгоритмы взвешенного рандома для выпадения карт и система Pity («гарант»).
  * Крафт мифических карт из осколков.
  * Интерактивные дуэли с расчетом шансов на основе силы карт и экипированных баффов.
  * Коллекции (сеты) с глобальными аурами и прогрессией (ежедневные стрики).
* **Социальный и административный функционал:**
  * **Глобальные и локальные лидерборды** (по балансу, картам и стрикам).
  * **Продвинутая система промокодов** (с лимитами по времени/использованиям и парсингом сложных JSON-наград).
  * **Монетизация:** Интеграция внутренней валюты Telegram Stars для покупки премиум-круток.
  * **UGC (Пользовательский контент):** Система предложки новых карт от игроков с квизом, модерацией и защитой от спама.
* **i18n (Локализация):** Полная поддержка русского и английского языков с динамическим определением языка.
* **Инфраструктура:** Автоматический деплой через **GitHub Actions** и ежедневные бэкапы БД с отправкой отчета в админ-чат.

## 🏗 Архитектура проекта

Проект структурирован в соответствии с принципами чистой архитектуры:

```text
├── cmd/bot/main.go            # Точка входа, DI-контейнер и инициализация
├── internal/
│   ├── delivery/              # Слой представления (роутеры Telegram и Discord)
│   ├── service/               # Слой бизнес-логики (Gacha, Duels, Suggest, Backup)
│   ├── repository/            # Слой данных (SQL запросы к PostgreSQL)
│   ├── models/                # Доменные сущности (User, Card, Rarity, Duel)
│   └── i18n/                  # Система локализации
├── locales/                   # JSON-словари для перевода
└── docker-compose.yml         # Инфраструктура (Бот, БД, Redis, Бэкапы)
```

## 🛠 Технологический стек

* **Язык:** Go (Golang)
* **База данных:** PostgreSQL 15 (драйвер `lib/pq`)
* **Кэш / In-Memory:** Redis 7 (клиент `redis/go-redis/v9`)
* **Telegram API:** `telebot.v3`
* **Discord API:** `discordgo`
* **Деплой:** Docker, Docker Compose, GitHub Actions (CI/CD)

## 🚀 Быстрый старт (Deployment)

Бот полностью упакован в Docker и готов к развертыванию на любом сервере.

### 1. Клонирование репозитория
```bash
git clone https://github.com/BasteArima/gachabot.git
cd gachabot
```

### 2. Настройка переменных окружения
Скопируйте пример конфига и заполните свои данные:
```bash
cp .env.example .env
nano .env
```
*Вам понадобятся токены от BotFather (Telegram) и Discord Developer Portal, а также ID вашего Telegram-аккаунта для админ-прав.*

### 3. Запуск
```bash
docker-compose up -d --build
```
Docker автоматически поднимет PostgreSQL, выполнит начальные миграции (`schema.sql`), поднимет Redis, сервис автоматических бэкапов и самого бота.
